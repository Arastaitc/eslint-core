## Install

Install the correct versions of package.

Yarn

```bash
yarn add @arastaitc/eslint-config --dev
```

Npm

```bash
npm install @arastaitc/eslint-config --save-dev
```

## Usage

```
// .eslintrc

{
  "extends": [
    "@arastaitc/eslint-config"
  ]
}

```
